# 🛣️ Highway 80 Dhaba — Official Website

> A fully responsive, modern website built for Highway 80 Dhaba, located on the Kalyan–Shilphata Highway, Mumbai. Built from scratch using pure HTML, CSS & JavaScript — no frameworks, no cost.

🌐 Live Demo

🔗 [sunil-0-shinde.github.io/highway80-dhaba](https://sunil-0-shinde.github.io/highway80-dhaba)

📌 About This Project

This website was designed and developed as a freelance project to give Highway 80 Dhaba a professional online presence. The business had no website despite having 578+ Google reviews and a 4.1★ rating. The goal was to build a clean, catchy, mobile-friendly site that works for the owner and attracts new customers.

---

✨ Features

- 🎨 Saffron & Green theme - Indian flag inspired colour palette
- 📱 Fully Responsive - Works on PC, Tablet, Android & iOS
- 🍽️ Interactive Menu - 4 tabs: Starters, Main Course, Tandoor, Drinks
- 📺 Vibe Section - Highlights live sports, BYOB, highway location, family seating
- 🖼️ Photo Gallery - 6-cell grid layout, ready for real images
- ⭐ Real Google Reviews - 3 verified guest reviews displayed
- 💬 WhatsApp Button - Direct chat with pre-filled message
- 🗺️ Google Maps Link - One tap to get directions
- 🌙 Scroll Animations - Smooth fade-up effects on all sections
- 🍔 Hamburger Menu - Mobile-friendly navigation

🚀 Deployment

This site is deployed for free using GitHub Pages.

Steps to deploy:
1. Push `index.html` to the `main` branch
2. Go to repo → Settings → Pages
3. Set source to `main` branch → `/ (root)`
4. Save - live in 1–2 minutes
